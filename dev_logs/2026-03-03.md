### 1.配置Unity3D及插件
#### 配置unity3d
```
https://unity.com/cn/releases/editor/archive
```
选择了2022.3.62.f3
```
tar -xf Unity-2022.3.62f3.tar.xz
```
进入该文件夹内部（通常是类似 `Unity-2022.3.62f3/Editor/` 的路径），找到里面那个名为 `Unity` 的可执行文件，选中并确认。

#### 配置untiy-ros通信插件
```
https://github.com/Unity-Technologies/ROS-TCP-Connector.git?path=/com.unity.robotics.ros-tcp-connector
```
这是Untiy Robotics Hub提供的插件

Unity 作为一个客户端发送/接收数据，而 ROS 2 端运行一个 TCP 服务端点（Endpoint）来中转数据。

在自己的工作空间内还需要下载配置ros tcp endpoints
```
git clone -b main-ros2 https://github.com/Unity-Technologies/ROS-TCP-Endpoint.git
```

启动通信端点
```
ros2 run ros_tcp_endpoint default_server_endpoint --ros-args -p ROS_IP:=0.0.0.0
```
